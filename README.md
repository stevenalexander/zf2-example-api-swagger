# ZF2 API documented by Swagger

This is an example application showing how to document a ZF2 RESTful API using [Swagger](https://github.com/zircote/swagger-php) and the [ZF2 Swagger module](https://github.com/outeredge/SwaggerModule).

## Requirements

* PHP 5.3+
* Web server [setup with virtual host to serve project folder](http://framework.zend.com/manual/2.2/en/user-guide/skeleton-application.html#virtual-host)
* [Composer](http://getcomposer.org/) (manage dependencies)

## Setup

1. Get composer:

    ```
    curl -sS https://getcomposer.org/installer | php
    ```

2. Install the dependencies:

    ```
    php composer.phar install
    ```
